# ESP32 LED Control

Control ESP32 LED using Streamlit web dashboard.

## Features
- LED ON/OFF control
- Web-based dashboard
- ESP32 WiFi control

## Files
- LED_WIFI_GLOW.ino → ESP32 code
- app.py → Streamlit dashboard
- requirements.txt → Python libraries

## Run

pip install -r requirements.txt

streamlit run app.py
